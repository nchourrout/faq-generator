# Website FAQ Generator

Generates the FAQs for any website using [Firecrawl](https://firecrawl.dev) `/extract` endpoint.

![Demo](./demo.gif)


## Installation

```bash
npm install
cp .env.example .env 
sudo nano .env # Add your Firecrawl API key
npm run dev
```
